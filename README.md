# Hard Truck Apocalypse: RenPy

</div>

**Hard Truck Apocalypse: RenPy** - фанатский перенос сюжета и некоторых элементов геймплея оригинальной игры на движок для визуальных новелл **RenPy**. 

Проект находится в активной разработке и не завершён на 100%.

<div align="center">

**[Отказ от ответственности](https://github.com/stakanyash/hta_renpy/blob/main/DISCLAIMER.md)**

</div>

---

### Что доступно:

- Сюжет первого региона по ветке отказа Лисе
- Случайные и сюжетные бои, которые реализованы по системе пошаговых RPG игр
- Система инвентаря для продажи предметов
- Покупка нового автомобиля (через диалог)

### В разработке:

- Сюжет первого региона по ветке согласия помочь Лисе
- Экран покупки автомобиля
- Экран покупки оружия
- Возможность устанавливать более мощное оружие на совместимые автомобили
- Полный сюжет всех четырёх регионов
- Все типы встреч (энкаунтеры), присутствующие в оригинальной игре
